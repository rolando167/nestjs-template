## 🌐 Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install foobar.

```c
#npm 📟
> npm i
> npm --version
> npm run lint

> npm install --save @nestjs/swagger
> npm i --save class-validator class-transformer

> npm install prisma --save-dev
> npx prisma init

> npm i --save @nestjs/config
```

## 🔰 Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
